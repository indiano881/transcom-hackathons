# 运行
该项目是一个前后端分离的 Web 项目，需要在两个终端窗口中分别运行后端代码和前端代码，至于它们分别如何运行可参考文档 backend/README.md 和 frontend/README.md，你需要先启动后端系统，然后再启动前端系统。

# 测试
当你分别成功运行了后端和前端项目，那么你现在可以在浏览器中通过地址 [http://localhost:5173/](http://localhost:5173/) 来访问该项目。

为了方便测试，本项目根目录下有如下三个 zip 文件：
- test-site.zip: 这是一个纯 HTML、CSS、JavaScript 的网站源码文件
- pure-html-website.zip: 这是一个纯 HTML、CSS、JavaScript 的网站源码文件
- react-website.zip: 这是一个使用当前最新版本的 [React](https://react.dev/) 开发的网站源码文件，React 是目前流行的前端框架之一
- vue-website.zip: 这是一个使用当前最新版本的 [Vue.js](https://vuejs.org/) 开发的网站源码文件，Vue.js 是目前流行的前端框架之一
